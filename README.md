# Estructuras de Control en JavaScript

Este proyecto demuestra el uso de condicionales (if/else) y ciclos (for) en JavaScript utilizando Node.js.

## Funcionalidad
- Recorre una lista de estudiantes
- Determina quién aprueba o reprueba
- Calcula el promedio del curso

## Cómo ejecutar
1. Abrir la terminal
2. Ejecutar:

node app.js
